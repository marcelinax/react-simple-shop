Simple react shop application

### Setup

How to setup and run project

#### Requirements

- Node 14.x+ (best 16.x+)

#### .env file

```
# Api/server url
REACT_APP_API_URL = https://reasonapps-gql-api.vercel.app/api/graphql
```

#### Installation and dev server

Development server will start on default port (:3000)

```
npm install
npm start
```
